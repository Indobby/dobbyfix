Service Codes Kawasaki

Code Error 11 เซ็นเซอร์คันเร่งทำงานผิดปกติ สายไฟถลอกหรือชำรุด
Code Error 12 เซ็นเซอร์ความดันอากาศเข้าทำงานผิดปกติ สายไฟถลอกหรือชำรุด
Code Error 13 เซ็นเซอร์อุณหภูมิอากาศเข้าทำงานผิดปกติ สายไฟถลอกหรือชำรุด
Code Error 14 เซ็นเซอร์อุณหภูมิน้ำทำงานผิดปกติ สายไฟถลอกหรือชำรุด
Code Error 21 เซ็นเซอร์เพลาข้อเหวี่ยงทำงานผิดปกติ สายไฟถลอกหรือชำรุด หลุดออกจากจุดยึด
Code Error 24 เซ็นเซอร์ล้อหลังทำงานผิดปกติ สายไฟถลอกหรือชำรุด
Code Error 25 เซ็นเซอร์ตำแหน่งเกียร์ทำงานผิดปกติ สายไฟถลอกหรือชำรุด
Code Error 31 เซ็นเซอร์ขาตั้งทำงานผิดปกติ สายไฟถลอกหรือชำรุด
Code Error 33 เซ็นเซอร์ออกซิเจนทำงานผิดปกติหรือหยุดทำงาน สายไฟถลอกหรือชำรุด
Code Error 39 เกิดข้อผิดพลาดในการสั่งการของกล่อง ECU
Code Error 41 หัวฉีดน้ำมันเชื้อเพลิง #1 ทำงานผิดปกติ สายไฟถลอกหรือชำรุด หลุดจากจุดยึด
Code Error 42 หัวฉีดน้ำมันเชื้อเพลิง #2 ทำงานผิดปกติ สายไฟถลอกหรือชำรุด หลุดจากจุดยึด
Code Error 51 คอยล์สติ๊ก #1 ทำงานผิดปกติ สายไฟถลอกหรือชำรุด
Code Error 52 คอยล์สติ๊ก #2 ทำงานผิดปกติ สายไฟถลอกหรือชำรุด
Code Error 56 รีเลย์พัดลมหม้อน้ำทำงานผิดปกติ สายไฟถลอกหรือชำรุด
Code Error 67 เครื่องทำความร้อนเซ็นเซอร์ออกซิเจนทำงานผิดปกติ สายไฟถลอกหรือชำรุด
